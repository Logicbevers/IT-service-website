# SEO Strategy

## Metadata

We use Next.js Metadata API for managing SEO tags.

- **Global Metadata**: Configured in \`src/app/layout.tsx\`.
- **Page Metadata**: Each page exports a \`metadata\` object or \`generateMetadata\` function.
- **OpenGraph**: Default OG images and descriptions are set globally and overridden per page.

## Sitemap & Robots

- \`sitemap.xml\`: Automatically generated by \`src/app/sitemap.ts\`. Includes all static and dynamic routes (fetched from Sanity).
- \`robots.txt\`: Generated by \`src/app/robots.ts\`. Allows indexing of all pages except \`/api/\`.

## Performance

- **Images**: All images use \`next/image\` for automatic optimization and lazy loading.
- **Fonts**: \`next/font\` is used to load Inter font with zero layout shift.
- **Static Generation**: Most pages are statically generated (SSG) or incrementally static regenerated (ISR) for optimal performance.
